# 응급 환자 분류 예측 실습 (Medical Triage)

환자 데이터를 활용하여 응급 환자 분류를 예측하는 머신러닝 실습 구현 코드입니다.

## 폴더 구성 안내

- **`synthetic_medical_triage.csv`**: 실습에 사용되는 병원 응급 환자 분류 데이터셋 파일입니다.
- **실습코드 1**: 다층 퍼셉트론 (MLP), 의사결정나무 (Decision Tree), k-최근접 이웃 (k-NN) 모델을 구현하고 학습하는 코드입니다.
- **실습코드 2**: 그래디언트 부스팅 (Gradient Boosting) 모델을 활용하여 성능을 개선하는 코드입니다.
